Creates an AppImage widget with the specified source.

The source parameter is the path or URL of the image to be displayed. The size parameter is the desired size of the image. The fit parameter specifies how the image should be inscribed into the space allocated for it. The color parameter is used to apply a color filter to the image. The colorBlendMode parameter determines how the color should be blended with the image. The alignment parameter specifies how the image should be aligned within its container. The colorFilter parameter is used to apply a color filter to the image. The placeholderBuilder parameter is used to build a placeholder widget while the image is being loaded. The semanticsLabel parameter is used to provide a description of the image for accessibility purposes. The errorBuilder parameter is used to build a widget when the image fails to load.
