dart_fusion library

A library that brings together a harmonious blend of essential tools, utilities, and components designed to supercharge your Dart projects.

Classes

A class for initializing DartFusion auto generation.

A widget that builds its child using a custom builder function with optional data.

A custom button widget that wraps the InkMaterial widget to provide tap interactions with various customization options.
DImage<Source extends Object>
A widget for displaying vector or bitmap images from different sources.
DLog
A simple logging utility for printing log messages with customizable log levels.
DModel
Base dart model which consist copyWith, toJSON, fromJSON and toString value;
DOverlay
A builder widget that displays an overlay.
DOverlayController
A custom overlay that displays a widget on top of other content.
DParse
A utility class for parsing mostly related to http request.
DStateInherited
A scope to pass DStateValue data to its descendant.
DStateValue
State child class of DStateWidget.
DStateWidget
Making StatefulWidget prettier and less boilerplate.
DTextArea
A custom text area widget.
InkMaterial
A custom widget that combines Material and InkWell to provide ink splash effects on taps.
Model
Annotation of class as an indicator to generate a fromJSON, toJSON and copyWith inside the annotated class.
ResponseModel
Basic model in root of every Response, containing success status, message and also data that extends DModel class.
Variable
Annotation of variable inside a class with Model annotation.
