Creates an DBuilder widget with the specified builder function and optional data.

The data parameter is an optional JSON object that can be used to pass data to the builder function. The builder parameter is a required function that takes a BuildContext and JSON data, and returns a Widget.

Example:

DBuilder(
  data: {"name": "John", "age": 30},
  builder: (context, data) {
    final name = data.of<String>("name");
    final age = data.of<int>("age");
    return Text("My name is $name and I am $age years old.");
  },
)
